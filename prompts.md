# prompts

## Commit Messages

- Write messages concisely in English
- Line 1: Summary, Line 2: Empty line, Line 3 onwards: Details
- Follow Conventional Commits with prefixes:
  - `feat:` Feature additions
  - `fix:` Bug fixes
  - `refactor:` Refactoring. Changes that don't add features or fix bugs
  - `chore:` Comment fixes, formatting, documentation changes that don't modify code
  - `test:` Adding or modifying test code
  - `debug:` Debug prints etc., code to be removed later
